Base Building Guide

Building a base is key to long-term survival in Subnautica 2. This guide covers everything from your first room to advanced multi-biome complexes.

Getting Started: Habitat Builder

Before you can build anything, you need the Habitat Builder tool:

ComponentMaterials
Habitat BuilderComputer Chip + Wiring Kit + Titanium
Computer Chip2x Copper Wire + 1x Gold + 1x Silver
Wiring Kit2x Silver
Tip: You can find Habitat Builder fragments near Lifepod wreckage and in small crates around the Safe Shallows.

Core Modules

These are the building blocks of any base. Each module has a specific purpose:

ModuleCostPurpose
Basic Compartment2x TitaniumStandard room. Connect multiple to expand.
I-Compartment2x TitaniumAllows T-shaped connections between corridors.
L-Compartment2x Titanium90-degree corner piece.
Large Room6x TitaniumBig open space. Good for indoor farms and storage.

Power Systems

Every base needs power. Without it, your Fabricator, lights, and oxygen won't work.

Power SourceOutputBest For
Solar Panel75W (day), 0W (night)Shallow bases (0–100m). Easy to build early.
BioreactorUp to 500WMid-depth bases. Feed with flora/fauna.
Thermal Plant250W per plantNear heat sources (vents, lava). Set and forget.
Nuclear Reactor500W per rodDeep bases. Needs uranium. Long-term power.
Power Strategy: Start with Solar Panels. Add a Bioreactor once you find Multipurpose Room fragments. Upgrade to Thermal or Nuclear for deep bases.

Hull Integrity

Every module you add affects hull integrity. If it drops below 0, your base floods.

  • Each corridor reduces integrity slightly
  • Large rooms and Moon Pools reduce integrity more
  • Reinforcements increase integrity (+7 each)
  • Foundations increase integrity (+2 each)
  • Going deeper reduces integrity faster
Common Mistake: Building a large base too deep without reinforcements. Always add Reinforced Walls when expanding at 200m+ depth.

Recommended Base Layouts

Early Game: The Starter Base

Keep it simple. Build in the Safe Shallows near your Lifepod:

  1. 1x Basic Compartment (main room)
  2. 1x Hatch (entrance)
  3. 2x Solar Panel on roof
  4. Fabricator + Medical Kit Fabricator inside
  5. 1x Locker for storage

Total cost: ~8 Titanium + 2 Quartz + wiring materials

Mid Game: The Research Base

Once you find the Multipurpose Room fragment:

  1. 1x Multipurpose Room (center)
  2. 1x Scanner Room (attached)
  3. 1x Moon Pool (for vehicles)
  4. 1x Bioreactor (power)
  5. Interior Grow Beds for food
  6. Wall Lockers for organized storage

Late Game: The Deep Outpost

For exploring deep biomes (300m+):

  1. Thermal or Nuclear power
  2. Multiple Reinforced Walls
  3. Moon Pool with Vehicle Upgrade Console
  4. Scanner Room with HUD chip
  5. Water Filtration Machine
  6. Modification Station

Co-op Base Building Tips

  • Shared building — all players can modify the same base
  • Designate roles — one person builds, another gathers materials
  • Multiple bases — build satellite outposts at different biomes for fast travel
  • Communication — use beacons to mark planned base locations
